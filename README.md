# NBA-Chatbot
A line chatbot built to report NBA live stats

## Installation and Building

```bash
# clone repository
git clone https://github.com/johnhckuo/NBA-Chatbot.git
cd NBA-Chatbot

# install required packages and dependencies
npm install

# build and run locally
npm run dev

```
